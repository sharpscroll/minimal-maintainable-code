# Installation

## Use in every Codex project

Copy the folder `minimal-maintainable-code` to:

```text
~/.agents/skills/minimal-maintainable-code/
```

## Use only in one repository

Copy the folder to:

```text
<repository>/.agents/skills/minimal-maintainable-code/
```

## Invoke it

In a coding task, select or mention:

```text
$minimal-maintainable-code
```

Then provide the task normally.

Example:

```text
$minimal-maintainable-code

Fix the calculator rounding issue and add a regression test.
```
