name: minimal-maintainable-code
description: Guide AI coding agents to build, modify, review, debug, or refactor code with the smallest clear and maintainable solution that fully satisfies the request. Use to prevent AI-generated code bloat, over-engineering, speculative abstractions, unnecessary dependencies or configuration, unrelated refactoring, and unverified completion while preserving correctness, readability, security, accessibility, tests, and output quality.
